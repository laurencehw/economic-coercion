## Troubleshooting

**If you see authentication errors on future pushes**:
- GitHub requires Personal Access Token (PAT) for HTTPS
- Go to GitHub Settings > Developer settings > Personal access tokens
- Generate token with `repo` scope
- Use token as password when prompted

**If you want to work from another computer**:
```bash
git clone https://github.com/laurencehw/economic-coercion.git
cd economic-coercion
git checkout develop
```

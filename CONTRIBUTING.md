# Contributing

This is primarily a single-author academic project, but feedback, corrections, and suggestions are welcome.

## How to Contribute

### Reporting Issues

Open a GitHub Issue for:

- **Factual errors** — Incorrect data, outdated information, or misrepresented sources
- **Typos and formatting** — Spelling, grammar, or rendering issues
- **Broken links** — Dead URLs or missing references
- **Suggestions** — Ideas for additional content, case studies, or improvements

### Pull Requests

Small corrections (typos, broken links) are welcome as pull requests. For larger changes, please open an issue first to discuss.

## Style Guidelines

### Writing

- Professional but accessible tone
- Active voice preferred
- Define acronyms on first use in each chapter
- Use "the United States" rather than "we" for analytical objectivity

### Citations

- Cite all factual claims
- Include URLs for online sources
- Note access dates for web sources

### Figures

Generated with R/ggplot2 using the custom theme in `R/setup_theme.R`:

```r
source("R/setup_theme.R")
# ... create plot ...
save_econ_figure("figures/fig_XX_YY_name.png", width = 10, height = 6)
```

## Code of Conduct

Be respectful and constructive. This project aims to provide balanced, multi-perspectival analysis of complex geopolitical topics.

## Questions?

Open an issue on GitHub.

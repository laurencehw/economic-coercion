# Data Visualizations and Figures Plan

**Last Updated:** 2025-01-19

## Overview

This document catalogs all planned data visualizations, maps, charts, and diagrams for the Economic Coercion textbook. Each chapter should include 5-8 high-quality visuals that bring data to life and help readers understand geographic, economic, and strategic dependencies.

**Visual Design Principles:**
- Clear, publication-quality graphics
- Consistent color scheme and styling across all chapters
- Source citations on every figure
- Accessible to colorblind readers
- Both print and digital-friendly formats

## Tools and Software

**Data Processing:**
- Python (pandas, matplotlib, seaborn) for data analysis and chart generation
- R (ggplot2) for publication-quality statistical graphics

**Mapping:**
- QGIS or ArcGIS for geographic visualizations
- Mapbox/D3.js for interactive maps (digital edition)

**Diagramming:**
- draw.io / Lucidchart for flow diagrams and conceptual frameworks
- Adobe Illustrator or Inkscape for final polish

**Data Sources - Quick Reference:**
- Trade: UN Comtrade, USITC DataWeb, OECD
- Energy: IEA, EIA, BP Statistical Review
- Minerals: USGS Commodity Summaries
- Semiconductors: TechInsights, SIA, Gartner
- Finance: SWIFT, BIS, IMF, World Bank
- Sanctions: OFAC, Treasury, research databases
- Technology: Patent databases, academic publications, company reports

## Quality Control Checklist

For each figure:
- [ ] Data sources clearly cited
- [ ] Title is descriptive and informative
- [ ] Axes labeled with units
- [ ] Legend included where necessary
- [ ] Color scheme consistent with book style
- [ ] Accessible to colorblind readers (use patterns/shapes in addition to colors)
- [ ] High resolution (300 DPI minimum for print)
- [ ] Figure number and caption follow book format
- [ ] Data validated against multiple sources where possible
- [ ] Aligns with narrative in surrounding text
